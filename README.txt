Smart Study Planner (HTML + CSS + JS)

How to run on phone (Android):
1) Install "Acode" from Play Store.
2) Extract this zip into a folder on your phone (e.g., Downloads/smart-study-planner).
3) In Acode: Menu → Open folder → select that folder.
4) Open index.html and tap the play/preview icon to run it.
5) Add tasks, check them off, and your data will stay using localStorage.

How to deploy (short):
- Create a GitHub repo, upload index.html, style.css, script.js.
- In repo Settings → Pages → set Branch: main, folder: / (root). You'll get a public URL.
